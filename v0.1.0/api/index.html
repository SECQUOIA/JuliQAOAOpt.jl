JuliQAOAOpt.qiskit_initial_parameters — Function
qiskit_initial_parameters(sampleset::SampleSet)
qiskit_initial_parameters(metadata::AbstractDict)

Return QiskitOpt-ready QAOA initial parameters from JuliQAOAOpt sample-set metadata. The order is beta values followed by gamma values, with gamma values converted from normalized JuliQAOA energies back to the original QUBO energy scale.
